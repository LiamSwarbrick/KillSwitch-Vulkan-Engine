Do the same for all uniform data, put it all in a big buffer you can 
index by Instance_ID, and boom, 1 descriptor set.
Called "Bindless" rendering.

YOU choose how you want to format your big ass uniform buffer.
You map the memory, and write to it in your format of choice.
And now on top of that, your drawing has no descriptor set bind calls
in between draw commands.
So all of your per object configuration goes in it.

Now any part of your engine can write to that buffer,
it's vulkan agnostic so any part of the engine could if it made sense.


Same for Draw Call generation:
- Put all the data in a buffer.
- Send it to the GPU.
